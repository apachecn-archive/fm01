# FMESC-1-C

胖蚊子电调 (FMESC-1) MOS 驱动小板.
与 [FMESC-1-A](../fmesc-1-a/), `FMC-1` 配合使用.

+ 设计电池电压: 2s ~ 4s  (6.6V ~ 16.8V)

+ 设计最大单路电流: 40A


## PCB 设计参数

| 序号 | 项目 | 数值 |
| :--: | :-- | ---: |
| 1 | PCB 大小 | 12x20mm |
| 2 | 材质 | FR-4 厚度 1.0mm 铜箔厚度 1oz (35um) |
| 3 | 层数 | 2 (双面) |
| 4 | 最小线宽线距 | 0.16mm (6mil) |
| 5 | 最小过孔内径 | 0.3mm |
| 6 | 最小过孔外径 | 0.6mm |
| 7 | 最小字符高度 | 0.8mm |
| 8 | 最小过孔间距 | 0.4mm |
| 9 | 最小过孔与线间距 | 0.25mm |
| 10 | 最小非金属化槽宽度 | 0.8mm |

推荐加工厂: [嘉立创](https://www.jlc.com/), [捷配](https://www.jiepei.com/)
